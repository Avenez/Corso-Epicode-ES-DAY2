Come creare un Template per un Post

di "Mario Rossi" - pubblicato il 25 ottobre 2020

  • Sudduvisione in blocchi:

    • La prima cosa da fare è suddividere in blocchi la pagina, identificando le parti comuni
    da applicare a tutte le pagine dove le varie aree saranno quindi suddivise in:

    • Contenuti di intersezione del tag <header>
    • Contenuti centrali variabili del tag <main>
    • contenuti di fine pagina comuni da inserire nel tag <footer>

  • Formattazione del codice:

    • Il testo dellarticolo dovrà essere formattato per identificare tutte le parti da selezionare con CSS.
    Perciò i passi saranno:

    1. Progettare il markup assegnando ad ogni stecifico testo il tag appropriato
    2. Scrivare l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
    3. Applicare gli stili CSS

    L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale

La diffusione di CSS nel tempo:

Anno % Di utilizzo Supporto Browser

Un test di validazione potrà essere fatto tramite il tool online dedicato allo scopo.
